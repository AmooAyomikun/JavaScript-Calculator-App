Calculator App (JavaScript) A fully functional calculator built using HTML, CSS, and Vanilla JavaScript. This project was created as a learning exercise to understand DOM manipulation, event handling, state management, and expression logic in JavaScript — without using any frameworks or libraries.

Features Basic arithmetic operations: Addition (+) Subtraction (-) Multiplication (*) Division (/) Decimal number support (.)

Expression display (e.g. 5+6*2) Chained calculations Example: 5 + 6 = 11 → + 4 = 15 Delete button (D) – removes last character Clear button (C) – resets calculator Input validation (prevents invalid expressions like ++, **, ..)

Real-time expression building Keyboard support: Numbers 0–9

Operators + - * / Enter → = Backspace → Delete

Smart decimal handling (only one decimal per number)

Continuous calculation flow (like a real calculator)
